🚀 Multi-Agent AI Resume Screening System – Deployed Successfully!

Excited to share the deployment of a real-world AI recruitment system built during my internship at Innomatics Research Labs under the guidance of Yasoda Sreeram Kalluri, Sir.

🎯 What makes this project unique?

This system leverages a multi-agent architecture with tool calling, where agents dynamically use external tools like Wikipedia and DuckDuckGo to enhance decision-making—similar to a real hiring process.

🔍 Key Highlights

✅ Skills Matching Agent – uses Wikipedia to understand and compare skills
✅ Experience Evaluation Agent – uses DuckDuckGo search to analyze real-world job expectations
✅ Salary Intelligence Agent – estimates market salary ranges
✅ Supervisor Agent – orchestrates all agents and produces final hiring decision

🛠️ Tech Stack

Python
Streamlit (UI & deployment)
LangChain (multi-agent + tool calling)
Groq LLM

⚡ Key Learning
Built a tool-calling multi-agent system using real-world data sources
Solved deployment challenges across environments (local → cloud)
Optimized agent workflows to reduce latency and avoid redundant or recursive processing loops
